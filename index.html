Discord Server Widget

Widget được nhúng bằng iframe — nếu không hiện, kiểm tra cài đặt server.

Lưu ý & kiểm tra:
  • Để widget hiển thị, chủ server phải bật Server Widget trong Server Settings → Widget.
  • Nếu iframe trống hoặc báo lỗi, thử mở link widget trực tiếp: Mở widget.
  • GitHub Pages / một số host có thể chặn sandboxed frames — nếu gặp vấn đề, host trên Netlify/ Vercel hoặc dùng reverse proxy.
